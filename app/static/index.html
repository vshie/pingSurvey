To create log files for simulation:

  1. Click "Start" to begin a survey session
  2. Collect some data (GPS and ping readings)
  3. Click "Stop" to end the session
  4. Return here to select the created log file

Log files are stored in: /usr/blueos/extensions/ping-survey/

Note: Simulation will play the selected file at 5x speed and loop continuously until stopped.
